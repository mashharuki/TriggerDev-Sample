# TriggerDev-Sample
trigger.devの調査・検証用リポジトリ

## はじめ方

```bash
mkdir quickstart && cd quickstart
yarn init -y
npx trigger.dev@latest init
```

サンプルで動かす。

```bash
npx trigger.dev@latest dev
```
